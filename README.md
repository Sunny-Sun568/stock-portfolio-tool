# Stock Portfolio Analysis Tool (ATVI vs RBLX)

This is an interactive Python tool for ACC102 Track 4.
It helps users compare the performance of ATVI and RBLX stocks.

## What it does
- Adjust how much money you put into ATVI vs RBLX
- Shows the average return, risk, and Sharpe ratio
- Draws a chart comparing the stock prices

## How to use
You can try it online here:
[Click here to open the tool](https://stock-portfolio-tool-jnxrxcd9bfvhdfazjvplhh.streamlit.app)

To run it on your own computer:
1. Install Python and the required libraries
2. Run: streamlit run app.py

## Files in this project
- app.py: The main code
- RBLX_ATVI.csv: The stock price data
- requirements.txt: The list of tools the code needs
